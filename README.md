Quality Mart Grocery E-Commerce

Quality Mart is a modern grocery e-commerce web application built to provide customers with a seamless online shopping experience. The platform allows users to browse grocery products, search items, explore categories, add products to the cart, and place orders with ease. The website features a clean and responsive user interface optimized for both desktop and mobile devices.

Features

- Product catalog with categories
- Product search functionality
- Shopping cart management
- Responsive mobile-friendly design
- UPI and Cash on Delivery support
- WhatsApp order integration
- Fast delivery highlights and promotional banners
- Secure checkout experience

Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Modern Responsive UI Components

Getting Started

Installation

npm install

Run Development Server

npm run dev

Build for Production

npm run build

Preview Production Build

npm run preview
